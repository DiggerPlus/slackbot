# Slackbot for diggerplus team
Slack bot for diggerplus team, build with [slackbot](https://github.com/lins05/slackbot).

# Usage

1. **`!help`**: Get help
2. **`!set (\w+) (.*)`**: Set keyword
3. **`!command`**: Get keyword
